# slack_bot
Slack bot codes



#Usage
##Webhook
1. generate webhook url (channel or DM)

    `https://api.slack.com/apps/<Channel ID>/incoming-webhooks?`

2. manage to webhooks dicts: run webhook.py


##Post
use **post_to_slack(message, webhook_url)** in /bot/post.py
